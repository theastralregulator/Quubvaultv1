# **App Name**: Quub

## Core Features:

- Hybrid Talent Discovery: Unified dual-mode browsing for 'Work' and 'Workers' with high-performance search and smart filters tailored for mobile views.
- Smart-Match AI Tool: An AI-powered matching tool that uses reasoning to recommend projects to freelancers and candidates to employers based on profile affinity and skill sets.
- Omnichannel Authentication: Seamless onboarding via Google, Email/Password, or Phone + OTP with role selection (Worker, Employer, or Both) stored in Firestore.
- Real-time Direct Messaging: Integrated chat system utilizing Firebase Realtime Database with presence indicators and file sharing capabilities.
- Geolocated Listings: Proximity-aware searching for 'Work' and 'Workers' to connect local opportunities using browser location data.
- Urgent Hire Workflows: Express posting option for critical tasks requiring rapid turnaround, highlighted with visual status badges.
- Verified Trust Engine: Trust system featuring verification badges, user availability status, and automated skill rating aggregation stored in a scalable database.

## Style Guidelines:

- Primary Color: Deep Purple (#6C2BD9), evoking a premium, high-tech startup feel suitable for professional networking.
- Background: Ultra-light Lilac Grey (#F8F7FB), keeping the mobile-first interface airy and clean while remaining on-brand.
- Accent: Electric Indigo (#475CEE), an analogous partner to the primary purple that adds high-contrast punch for CTAs and highlights.
- Font pairing: 'Poppins' for bold, modern headlines and 'Inter' for clear, objective body text. These provide the 'startup-level' feel requested.
- Mobile-first card architecture with rounded edges (16px) and high-elevation shadow play to simulate a high-end application experience.
- Smooth micro-interactions on hover and soft fade transitions between navigation states to ensure the platform feels 'premium'.
- Thick-lined geometric icons to provide clarity and aesthetic consistency on smaller mobile touchscreens.
